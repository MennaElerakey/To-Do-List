
# Simple To-Do List Application

A beginner-friendly Python console application that allows users to manage daily tasks easily through a simple menu-based interface.

## Features
- Add new tasks
- View all existing tasks
- Mark tasks as completed
- User-friendly console interaction

## How It Works
The program runs in a loop and displays a menu with four options.  
Users can choose an option by entering a number from 1 to 4.

## Technologies Used
- Python 3
- Basic concepts: lists, loops, conditionals, user input

## How to Run
1. Make sure Python 3 is installed on your system.
2. Download or clone this repository.
